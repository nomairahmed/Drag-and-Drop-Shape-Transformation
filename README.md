# Drag-and-Drop-Shape-Transformation
Small webpage with transforming shapes
## Run Instructions
To run simply pull the repository and launch index.html
## Tech stack choice and justification
I opted for vanilla javascript for this task as it was intended to be a simple task and was acheivable in the time provided using vanilla javascript. I made use of libraries available to html, css and js such as drag and drop api, DOM manipulation and clip path for the shape. having no external dependencies make the project easy to run
## Trade offs due to time
I was able to acheive everything for the core task, however the bonus tasks such as Content Security Policy, Accessibility and Unit Tests were not acheived
